# Official AT Protocol interoperability fixtures

The files in `syntax/` and `crypto/` are copied unchanged from
[`bluesky-social/atproto/interop-test-files`](https://github.com/bluesky-social/atproto/tree/a9ff2da83b93e5211f45667e652f8a928f4dce29/interop-test-files)
at commit `a9ff2da83b93e5211f45667e652f8a928f4dce29`.

Only the formats implemented by the current hands-on are included. The upstream
specification remains authoritative if a fixture and specification disagree.
